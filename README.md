## 環境安裝指南

### 使用 pip 安裝
請確保已安裝 Python，且版本需 **>= 3.8**。

1. Clone或下載本專案至本地端。
2. 在專案根目錄下執行以下指令，安裝所需套件：

   ```bash
   pip install -r requirements.txt
