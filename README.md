# 測試

本機器人建立於 **Python 3.11**，並使用 **conda** 建構，主要使用抓取網頁的Server ID，並藉由OCR辨識抓取，開發環境基於 **Windows**。

## 環境安裝指南

### 使用 pip 安裝
請確保已安裝 Python，且版本需 **>= 3.8**。

1. Clone 或下載本專案至本地端。
2. 在專案根目錄下執行以下指令，安裝所需套件：

   ```bash
   pip install -r requirements.txt
   ```

### 注意
請注意：需自行安裝環境，專案未包含完整打包的exe檔案，請在Python底下執行。
